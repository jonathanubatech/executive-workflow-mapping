# Workflow Diagnostic Checklist

## Qualification Rubric

Before investing in a workflow redesign, verify these 5 must-haves:

| # | Must-Have | Met? | Notes |
|---|----------|------|-------|
| 1 | One recurring workflow pain | [ ] | Can the team name a specific, recurring bottleneck? |
| 2 | One accountable owner | [ ] | Is there a real person (not "the team") who owns this? |
| 3 | Existing systems in place | [ ] | Are the tools already deployed (M365, Jira, etc.)? |
| 4 | Measurable cost | [ ] | Can you quantify hours/week or dollars wasted? |
| 5 | Budget path | [ ] | Is there authority for a pilot investment? |

**Rule:** If 3 or fewer are true, this workflow is not ready for a sprint.
